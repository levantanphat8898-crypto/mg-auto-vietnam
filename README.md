# MG Bình Dương

Landing page React + Vite cho đại lý MG Bình Dương, lấy cảm hứng từ [mg-auto-insight.lovable.app](https://mg-auto-insight.lovable.app/).

## Công nghệ

- React 19
- Vite 8
- Tailwind CSS 4

## Chạy dự án

```bash
npm install
npm run dev
```

Mở [http://localhost:5173](http://localhost:5173) trên trình duyệt.

## Build production

```bash
npm run build
npm run preview
```

## Cấu trúc

```
src/
├── components/   # Header, Hero, Products, Contact, ...
├── data/       # Dữ liệu xe, khuyến mãi
├── App.jsx
└── index.css
```
